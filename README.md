### ✅ 3. 과제 제출하기
실습이 끝나면 내 폴더의 내용을 교수님께 최종 제출해야 합니다.

1. 왼쪽 메뉴의 **가지 모양 아이콘**(Source Control)을 누릅니다.
2. 메시지 칸에 **"미션3-이름-완료"**라고 적고 **[Commit]** 버튼을 누릅니다.
3. 파란색 **[Sync Changes]** 버튼을 누르면 최종 제출됩니다!
